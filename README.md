# ToodleDo

![Toodle Preview Image](readme_assets/toodledo.png)

## About ToodlDo

ToodleDo is a todo list that allows you to have todo lists for your todo list.

## How to use

1. Double click or press to edit a todo item.

2. Add a todo  
   ![add row button](/readme_assets/addrow.png)
3. Add a sub todo list  
   ![add sub todo list button](/readme_assets/sublist.png)
4. Mark a todo completed  
   ![checkmark](/readme_assets/check.png)
5. Delete a todo  
   ![delete todo button](/readme_assets/delete.png)
